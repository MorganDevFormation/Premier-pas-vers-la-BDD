# Okanban E02

## Objectifs

Implémenter les routes suivantes :

- GET /lists/:id
- DELETE /lists/:id
- PATCH /lists/:id

## GET /lists/:id

- Dans le controller `list.controller.js`, ajouter la méthode `getById` pour récupérer une liste par son ID.
- Dans le router de `list.router.js`, ajouter la route pour `getById`
- Vérifier que la réponse est correcte pour les ID valides et non valides.

## DELETE /lists/:id

- Dans le controller `list.controller.js`, ajouter la méthode `deleteById` pour supprimer une liste par son ID.
- Dans le router de `list.router.js`, ajouter la route pour `deleteById`.
- Vérifier que la réponse est correcte pour les ID valides et non valides.

## PATCH /lists/:id

- Dans le controller `list.controller.js`, ajouter la méthode `updateById` pour mettre à jour une liste par son ID.
- Dans le router de `list.router.js`, ajouter la route pour `updateById`.
- Vérifier que la réponse est correcte pour les ID valides et non valides, avec toutes les données dans le body, puis partielles.
