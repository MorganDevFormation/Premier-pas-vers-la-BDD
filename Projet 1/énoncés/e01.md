# Okanban E01

## Objectifs

- Créer un *Hello World* en NodeJS + Express
- Créer les modèles pour l'ORM Sequelize
- Créer les tables de la BDD via Sequelize
- Seeding de la BDD via Sequelize

UTILISER un linter si besoin

## Hello World

- Créer un dossier `api` et aller dedans pour faire un init de projet node.
- Ajouter les dépendances qui seront nécessaires pour le projet, comme `express`, `sequelize`, `pg` pour PostgreSQL et d'autres.
- Créer un fichier `app.js` ou `index.js` et créer une app express retournant *hello world* sur la route `/`
- Vérifier son fonctionnement

## Model Sequelize

- Créer un dossier `models`
- Créer un fichier pour le client Sequelize : `sequelize.client.js`
- Créer une base de données `okanban` avec un utilisateur `okanban` et un mot de passe `okanban`.
- Créer les modèles pour les tables :
    - liste dans un fichier `list.model.js`
    - carte dans un fichier `card.model.js`
    - tag dans un fichier `tag.model.js`
- Créer un fichier `models/index.js` pour déclarer les associations et exporter les modèles

## Sync DB

- Créer un dossier `migrations`
- Créer un fichier `01.create-tables.js` pour synchroniser les modèles avec la base de données
- Ajouter le script `"db:create": "node migrations/01.create-tables.js",` dans `package.json`
- Lancer la commande `npm run db:create` pour créer les tables dans la base de données.
- Vérifier que les tables ont été créées avec succès.

## Seed DB

- Créer un fichier `02.seed-tables.js` dans le dossier `migrations`
- Ajouter le script `"db:seed": "node migrations/02.seed-tables.js",` dans `package.json`
- Lancer la commande `npm run db:seed` pour peupler les tables avec des données de test.
- Vérifier que les données ont été insérées avec succès.
