# Okanban E03

## Objectifs

Implémenter les routes suivantes :

- GET /cards
- GET /cards/:id
- POST /cards
- PATCH /cards/:id
- DELETE /cards/:id

## Init

Créer les fichiers nécessaires pour implémenter les routes de `card` (à l'image de `list`).

## GET /cards

- Dans le controller `card.controller.js`, ajouter la méthode `getAll` pour récupérer toutes les cartes.
- Dans le router de `card.router.js`, ajouter la route pour `getAll`.
- Vérifier que la réponse est correcte pour les requêtes valides.

## GET /cards/:id

- Dans le controller `card.controller.js`, ajouter la méthode `getById` pour récupérer une carte par son ID.
- Dans le router de `card.router.js`, ajouter la route pour `getById`
- Vérifier que la réponse est correcte pour les ID valides et non valides.

## POST /cards

- Dans le controller `card.controller.js`, ajouter la méthode `create` pour créer une nouvelle carte.
- Gérer la validation des données pour la création d'une carte.
- Dans le router de `card.router.js`, ajouter la route pour `create`.
- Vérifier que la réponse est correcte pour les données valides et non valides.

## PATCH /cards/:id

- Dans le controller `card.controller.js`, ajouter la méthode `updateById` pour mettre à jour une carte existante.
- Gérer la validation des données pour la mise à jour d'une carte.
- Dans le router de `card.router.js`, ajouter la route pour `updateById`.
- Vérifier que la réponse est correcte pour les données valides et non valides.

## DELETE /cards/:id

- Dans le controller `card.controller.js`, ajouter la méthode `deleteById` pour supprimer une carte par son ID.
- Dans le router de `card.router.js`, ajouter la route pour `deleteById`.
- Vérifier que la réponse est correcte pour les ID valides et non valides.

## BONUS
- Implémenter un middleware d'erreur 404 généraliste pour gérer les routes non trouvées.
